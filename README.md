# GGB Point of Sale (PoS)

This repository contains both the **backend (Flask)** and the **frontend (Flutter)** for the GGB Point of Sale system.  
It is designed to provide a complete solution for inventory, sales, expenses, and reporting.

---

## 📂 Project Structure

- `backend/` → Flask backend with authentication, database models, and REST APIs
- `gb_pos/` → Flutter mobile app (Android/iOS) with UI screens and services
